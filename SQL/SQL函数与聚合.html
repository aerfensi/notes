数学函数

  • ABS(x)：绝对值
  • MOD(x,y)：取模。x/y的余数
  • RAND([x]) 【MySQL】：0~1的随机数，给定种子，则返回一个0~1的数，种子不变，返回的值不变
  • FLOOR(x)：小于x的最大整数
  • CEILING(x) 【MySQL】：大于x的最小整数
  • CEIL(x)【Oracle】：同上

聚合函数

聚合函数会直接忽略NULL值。
聚合函数可以结合GROUP BY，用于分组中；不分组的话，就作用于所有数据。
聚合函数只能用在SELECT中，不能用在WHERE中。

  • AVG(col)：返回指定列的平均值。
  • COUNT(col)：返回指定列中非NULL值的行数（当函数参数为星号*时不会忽略NULL，即使该行的值全为NULL，也会被统计进行数中）。
  • MIN(col)：返回指定列的最小值。
  • MAX(col)：返回指定列的最大值。
  • SUM(col)：返回指定列的所有值之和。
  • GROUP_CONCAT(col)【MySQL】：返回由属于一组的列值连接组合而成的结果，多个数据之间用逗号隔开。

GROUB BY

与SELECT配合，把某一列中的相同的数据分为一组，同在一组的多行会被合并为1行。
合并后的行中，如果有列的数据不是全都相同的，取第一个数据。

  1. GROUB BY如果和WHERE一同使用，GROUP BY必须在WHERE后面。
  2. GROUP BY col1, col2 表示只有这两列的数据都相同时，才会被分到同一组
  3. 使用group by时，select的列名必须是group by后面跟的列名，或者是聚合函数
-- 正确
select id,count(CId) from sc group by id;
-- 错误
select * from SC group by id;

HAVING

WHERE不能和聚合函数一起使用，也就是说不能用WHERE对GROUP BY之后的结果进行筛选，所以才发明了HAVING。
HAVING只能用在GROUP BY后面。

xxxxxxxxxx
-- 下的订单的总金额小于2000的客户
SELECT Customer,SUM(OrderPrice) FROM Orders
GROUP BY Customer
HAVING SUM(OrderPrice)<2000;

字符串函数

  • CONCAT(str1,str2,...)：连接多个字符串
  • REVERSE(str)：反向字符串
  • TRIM(str)，LTRIM(str)，RTRIM(str)：去掉收尾空格，同Python str类型的strip()，lstrip()，rstrip()方法
  • RIGHT/LEFT(str,n)【MySQL】：取字符串右/左边的n个字符
  • lower(str)/upper(str)/nls_lower(str)/nls_upper(str)【Oracle】：nls这个支持包含非字母的字符串，它将只转换字母，而不去改变非字母。
  • length(str)【Oracle】
  • replace(string, search[, replaceStr])【Oracle】：不加replaceStr的话，就删掉找到的子串
  • substr(string, start[, length])【Oracle】：返回从start开始的长度的length的字符

日期函数

MySQL日期函数

time参数我一般都用NOW()函数的返回值，当然也可以是符合格式要求的字符串。
比如希望返回后一个时间点的分 SELECT MINUTE('3:15:32')。

  • NOW()：获取当前YYYY-MM-DD HH:MM:SS，也就是DATETIME类型的数据
  • DATE(time)：获取当前YYYY-MM-DD，也就是DATE类型的数据；CURRENT_DATE()【CURDATE()】，直接获取当前日期
  • TIME(time)：获取当前HH:MM:SS，也就是DATE类型的数据；CURRENT_TIME()【CURTIME()】，直接获取当前时间
  • YEAR(time)：获取当前YYYY，也就是YEAR类型的数据
  • MONTH(time)
  • DAY(time)
  • HOUR(time)
  • MINUTE(time)
  • SECOND(time)
  • MONTHNAME(time)
  • DAYNAME(time)

时间类型是支持加减运算的，比如计算宠物的年龄

xxxxxxxxxx
SELECT name, birth, CURDATE(),
    (YEAR(CURDATE())-YEAR(birth)) AS age
    FROM pet;

Oracle

sysdate，systimestamp：这两个Oracle自带的属性就是当前的系统时间，sysdate精确到秒，systimestamp精确到毫秒。
to_date(str, format) / to_char(date, format)：这两个函数可以转换字符串和date类型。

select TO_DATE('2018','YYYY') as t1,TO_CHAR(SYSDATE,'YYYY') as t2 from TEST;

Oracle decode

decode(左值, 右值/条件, 返回值1, 返回值2)
左值一般是通过select查询出来的字段，如果【左值】等于【右值】，或者【左值】满足【条件】，则返回值1，否则返回值2。
类似于三目运算符。

decode(左值, 右值1/条件1, 返回值1, 右值2/条件2, 返回值2, … 右值n/条件n, 返回值n, 缺省值)
类似于switch语句。也可以没有缺省值，这样就类似于if分支。

select * from tbl where name in ('bbb','aaa') order by decode(name, 'bbb', 1, 'aaa',2);
查询表格中name为aaa或者bbb的记录。in语句中的内容会自动排序，也就是说默认查询结果顺序是aaa的记录在bbb前面。
我希望查询结果根据in语句原有的顺序排序，所以使用了order by + decode的方式。
