引号的用法

mysql：单引号双引号都表示字符串，反单引号用在表格名或列名上，可以避免与SQL关键字同名引起的错误。
oracle：单引号表示字符串，双引号用在表格名或列名上，使其严格区分大小写。

MySQL char、varchar、text

char：定长，有默认值，长度不足用空格补足。
varchar：变长，可设置长度上限，有默认值。
text：变长，不可设置长度上限，没有默认值。

MySQL常用命令

status：显示当前connection的状态。
use db_name：切换数据库。
SELECT database();：查看当前数据库，用status命令也能查看。
CREATE DATABASE IF NOT EXISTS dbname;：创建数据库。
DROP DATABASE dbname;：删除数据库，直接删除当前数据库也不会有问题。
show databases;：显示所有数据库。
show tables;：显示当前数据库中的表格。

数据库事务

事务可以简单理解为一组SQL语句，这组SQL语句要么都执行，要么都不执行。

四个特性（ACID）
原子性：事务是一个整体，同一个事务中的操作，要么都执行，要么都不执行。
一致性：事务开始前，到事务结束后，数据库中的数据都要是一致的。
隔离性：多个事务并发执行，他们之间是互相隔离的，不能干扰。
持久性：事务结束后，它对数据库的修改是能够被永久保存的。

Oracle在第一次遇到DML语句（update、insert、delete）时，会自动开启事务。在提交或回滚后，事务结束。
提交：commit、遇到DDL语句（create、alter、drop）、正常断开Oracle连接。
回滚：rollback、程序异常终止。

MySQL默认是自动提交事务的，所以需要使用关键字begin来开启一个事务。

多表查询

内连接只会查询到两张表格中都存在的行。
左/右外连接，会把左/右表格中存在的行都查询出来，另一张表格中如果没有对应的数据，就用NULL填充。
