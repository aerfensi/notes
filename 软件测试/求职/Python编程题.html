判断三个数字能否组成等边三角形

xxxxxxxxxx
def func(a,b,c):
    # 最小的边小于0，则不能组成三角形
    if min(a, b, c) <= 0:
        return False
    # 任意两边之和大于第三边，能组成三角形
    # 即较小的两边之和大于第三边，能组成三角形
    if a + b + c - max(a, b, c) > max(a, b, c):
        # 判断是不是等边
        if a == b or a == c or b == c:
            return True
    return False

素数/质数

素数/质数：一个大于1的自然数，除了1和它自身外，不能被其他自然数整除的数叫做质数。
在[1，20]中，哪些是素数？

xxxxxxxxxx
def func():
    for i in range(1, 21):
        # 1 不是素数
        if i == 1:
            print(i, 'False')
            continue
        for j in range(2, i):
            if i % j == 0:
                print(i, 'False')
                break
        else:
            print(i, 'True')

1，2，3，4能组成多少互不相同且无重复数字的三位数

def func():
    l = [1, 2, 3, 4]
    r = [i * 100 + j * 10 + k for i in l for j in l for k in l if i != j and i != k and j != k]
    print(r)

判断字符串中有多少数字、字母

xxxxxxxxxx
def func():
    import re
    s = '1a 0 中bc 中'
    digit = 0
    alpha = 0
    space = 0
    others = 0
    for i in s:
        if i.isdigit():
            digit += 1
        elif i.isspace():
            space += 1
        elif re.fullmatch('[a-zA-Z]', i):
            alpha += 1
        else:
            others += 1
    print('digit:{} alpha:{} space:{} others:{}'.format(digit,alpha,space,others))

水仙花数

水仙花数：一个三位数，其各位数字立方和等于该数本身。例如：153是一个”水仙花数”，因为153=1的三次方＋5的三次方＋3的三次方。
输出[100,1000)之间的水仙花数。

xxxxxxxxxx
def func():
    for i in range(100, 1000):
        a, b, c = [int(j) for j in str(i)]
        if a ** 3 + b ** 3 + c ** 3 == i:
            print(i, a, b, c)
