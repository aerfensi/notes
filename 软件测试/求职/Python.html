Python

鸭子类型

鸭子类型就是不关心对象的具体类型，而是关心对象的功能。
比如当我需要一个迭代器的时候，我不关心对象的具体类型是什么，只要它实现了__iter__和__next__方法就行了。

GIL

GIL，即全局解释锁，是cpython的一个特性。他保证同一时间内只有一条线程在使用CPU。
这降低了多线程的效率，要避免这个问题可以使用多进程。

互斥锁和Gil锁的关系

Gil锁：保证同一时刻只有一个线程能使用到cpu。
互斥锁：多线程时,保证修改共享数据时有序的修改,不会产生数据修改混乱。

内存管理

Python中当对象增加一个变量名，或者被放入到一个容器中，或者被传给一个函数，会增加对象的引用计数；当变量名被del关键字销毁，或者被重新赋值，或者超出变量作用域，对象的引用计数会减少。
当对象的引用计数归零后，对象会被Python的垃圾回收器回收掉。Python还有循环垃圾回收器，用来回收由于相互引用导致他们的引用计数永远不会归零的对象。

匿名函数

lambda x,y : x+y
也就是lambda表达式。
函数体只能是单一语句，该语句的结果就是lambda表达式的返回值。
一般用法是将其作为参数传递给其他函数。

浅拷贝和深拷贝的区别

copy.copy(obj)->new_obj
copy.deepcopy(obj)->new_obj
浅拷贝创建一个新对象，该对象绑定的属性依旧指向原对象的属性。
深拷贝创建的新对象，新对象绑定的属性也是新创建出来的。

*args和**kwargs

*args：任意数量位置参数，args是一个元祖。他的后面只能是关键字参数。
**kwargs：任意数量关键字参数，kwargs是一个字典。他必须是最后一个参数。

魔术方法

魔术方法就是名字中带前后双下划线的方法。在类中实现的魔术方法会在特定的时候被调用，用来实现特定的功能。
比如init会在对象创建时被调用，用来初始化对象。

什么是切片

切片可以用来获取序列的指定部分。
切片获取的序列包含起始位置，不包含终止位置。
切片通过设置步长可以反向获取。
s[::-1]：反向整个序列。
s[1:5:2]：获取小标为1、3的元素。

dir()和dict

dir()：列出对象能访问的所有成员，包括类成员。
__dict__：列出与对象绑定的成员

变量命名

_value：在模块中定义私有变量，不会被from module import *导入。
__value：在类中定义变量，其真实名字会变成_classname__value，可以防止被子类中的成员覆盖

实例方法、静态方法、类方法

实例方法：通过对象调用，有self
静态方法，@staticmethod：通过类调用，没有self也没有cls
类方法，@classmethod：通过类调用，有cls

常见的异常类型

所有异常都继承自BaseException。
AttributeError：被访问对象的成员是不存在的。
IndexError：下标越界。
ImportError：导入的包或模块不存在。

迭代器和生成器

  1. iterable：实现了__iter__方法的对象。
  2. iterator：实现了__iter__和__next__方法的对象
  3. generator：生成器函数/生成器表达式返回的的对象
  4. generator function：带 yield 关键字的函数
  5. generator expression：(elem for elem in [1, 2, 3])
