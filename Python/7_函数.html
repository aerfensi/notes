函数成员

可以给函数对象动态绑定成员，当然一般不会去这么做
以下是一些函数的魔术变量

  • __name__
  • __qualname__：函数限定名称，ClassName.func_name
  • __globals__：函数所在模块的全局变量
  • __kwdefaults__：函数关键字参数的默认值
  • __closure__：闭包绑定的自由变量
  • __doc__
  • __annotations__：参数与返回值的注解

参数

位置参数：和其他编程语言一样，按参数定义的顺序传入参数，func('Tom',13)
关键字参数：传入参数时带参数定义时的名字，可以无视参数位置，func(age=13,name='Tom')
关键字参数后面必须都是关键字参数

默认参数：def func(i:int=1,j:int=2)->str:，默认参数可以不传入
任意数量位置参数：*args，这些参数会被组成一个元组；
任意数量关键字参数：**kwargs，这些参数被组成一个字典；

**kwargs必须是最后一个参数
*args或者*后面只能是关键字参数，它可以用来强制使用关键字参数，比如def a(x, *args, y)，y 只能用关键字参数的方式传入

参考原则：def func(name, age, sex=1, *args, **kargs)，但并不是说其他的顺序就会导致语法问题

坑1：修改默认参数

改变默认参数的话，在下次调用这个函数时，使用的默认参数就是这个被改变过的参数，而不是会重置这个参数。
python 是引用传递，函数得到的是引用的副本，所以改变引用的指向，并不会改变原引用。故改变默认参数的引用指向，并不会修改这个参数。
通过引用改变实例，才会有影响。

匿名函数（lambda 表达式）

add = lambda x, y: x + y
函数体是一个表达式，函数的返回值就是这个表达式的运算结果；当然也可以不是表达式，而是调用别的函数
lambda 和普通函数的类型是一样的，都是 function
lambda 中使用了外部变量和普通函数使用外部变量时的情况是一样的，在函数被调用时才会去取得外部变量的值，而不是定义时

lambda 函数不建议赋值给变量！，虽然 python 中一切都是对象，lambda 函数也是对象，但它是匿名函数，如果把它赋值，那就不是匿名，而是有名了。如果希望得到有名的函数，只能用常规方法定义函数。
lambda 函数的作用就是作为参数传给其他函数

闭包就是一个访问了外部函数局部变量（不可以是类的局部变量）的函数。
判断一个函数是不是一个闭包可以使用这个函数的__closure__属性，非闭包的函数，这个属性的值是 None，闭包的这个属性是一个元组，元组里的元素是 cell 类型的对象，每一个 cell 中保存了闭包访问的外部函数的局部变量。（保存在cell的cell_contents成员属性中，它仅仅是一个普通属性，不是列表或元组）
因为有这个__closure__在，所以闭包脱离了外部函数，还能访问外部函数的局部变量。
多次执行外层函数得到的闭包也不是同一个，就像是一个返回对象的函数被执行多次，返回的对象也是多个且互不相关。

闭包的应用

  1. Python 装饰器就是闭包
  2. 闭包可以用来简化类，类似于 lambda 表达式可以简化函数

常用高阶函数

高阶函数就是接受函数作为参数，或者能够返回一个函数的函数。比如说闭包的外部函数

sorted(iterable, key=None, reverse=False)->list
key 是一个只接受一个参数的函数，如果 key 不为 None，那 sorted 将根据 key 的返回结果对 iterable 进行排序
忽略大小写，对这些字符串排序
sorted(('bob', 'about', 'Zoo', 'Credit'), key=str.lower)

map(function, iterable, ...)->map
接受一个或多个 iterable，function 是一个只接受一个参数的函数，把 function 依次作用与 iterable 的所有元素。
返回的值组成一个 map 对象，这个 map 是一个 iterator。

把列表中的所有元素转成 str。
map(str, [1, 2, 3, 4, 5, 6, 7, 8, 9])->filter

filter(function, iterable)
同样让 function 作用域每个元素，根据 function 返回 True 或 False，决定是否保留该元素。
返回的 filter 对象是一个 iterator
filter(lambda x:x>5,[4,2,6,4,8,1,2,10])

functools.reduce(function,iterable)
functools.reduce(lambda a,b:a*b,range(1,4))
reduce 返回的是最后积累执行的结果，也就是最后一次执行 function 参数时返回的结果，不像 filter 和 map 一样返回 iterable

map 和 filter 可以用列表推导式代替。
map 和 filter 返回的其实是一个 generator，所以其实用生成器表达式代替更合理。

reduce 没有完全代替的方法，但是有一些函数可以提供简单的 reduce 功能

  • 内置sum函数，求和
  • operator.mul，求积函数，可以用来代替lambda a,b:a*b，依旧要配合reduce函数使用
  • operator.itemgetter，返回的是一个可执行对象，效果如下：
    After f = itemgetter(2), the call f(r) returns r[2].
    After g = itemgetter(2, 5, 3), the call g(r) returns (r[2], r[5], r[3]).
  • operator.attrgetter，和nametuple配合使用就可以根据名字取出属性值了
    After f = attrgetter('name'), the call f(b) returns b.name.
    After f = attrgetter('name', 'date'), the call f(b) returns (b.name, b.date).
    After f = attrgetter('name.first', 'name.last'), the call f(b) returns (b.name.first, b.name.last).

functools.partial(func, *args, **keywords)：返回一个可执行对象，用来固定 func 函数中的部分参数
obj=partial(func, 'img', cls='pic-frame')：固定函数 func 的第一个参数的值是img，cls 参数的值是pic-frame
partial 对象有 func、args、keywards 属性，可以访问原函数，位置参数，关键字参数

functools.partialmethod，同上，只不过它作用于方法而非函数
